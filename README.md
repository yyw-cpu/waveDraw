# WaveDraw

WaveDraw 是一款专业的波形绘制与分析软件，提供直观的波形可视化、数据处理和导出功能。

## 功能特性

- 波形可视化与绘制
- 数据导入与导出
- 多格式支持
- 专业分析工具
- 用户友好的界面

## 系统要求

- Windows 7/10/11
- 管理员权限（用于安装）
