# 贡献指南

感谢您考虑为 WaveDraw 做出贡献！

## 行为准则

- 尊重所有参与者
- 保持专业和友善的沟通
- 接受建设性的批评

## 如何贡献

### 报告问题

1. 在提交新问题前，请先搜索现有问题列表
2. 使用清晰、描述性的标题
3. 提供详细的复现步骤
4. 包含系统环境信息
5. 如有错误信息，请提供完整日志

### 提交代码

1. Fork 本仓库
2. 创建功能分支：`git checkout -b feature/您的功能名`
3. 提交更改：`git commit -m '添加某功能'`
4. 推送到分支：`git push origin feature/您的功能名`
5. 提交 Pull Request

### 代码规范

- 遵循现有代码风格
- 添加必要的注释
- 确保代码可维护性
- 不要提交敏感信息

## 知识产权声明

通过向本项目提交贡献，您同意：

1. 您拥有提交内容的知识产权，或有权提交
2. 您的贡献将在与项目相同的许可证下发布
3. 您授予项目维护者使用、修改和分发您贡献的权利
4. 您的贡献不会侵犯任何第三方的知识产权

## 审核流程

- 所有提交都需要维护者审核
- 可能需要多次修改才能合并
- 维护者保留拒绝任何贡献的权利

## 联系方式

如有疑问，请通过 Issues 或电子邮件联系维护团队。
